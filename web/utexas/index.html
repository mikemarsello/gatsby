UTexas

The UT homepage is currently built in Drupal 7 and hosted on Pantheon. I played a key role in developing the theme for this site, which is a custom theme using the Zurb Foundation front-end framework. There have been multiple redesigns over the years, particularly to the home page. One of the last things I did while working in the ITS department was to code the home page which uses many scroll effects.

UT Drupal Kit

The DrupalKit is a custom distribution which provides UT campus developers with a number of themed and accessible widgets along with a layout tool to easily build pages. There are a number of add-on content types to work with events, faculty profiles, and news/blog articles. The codebase is supported on Pantheon hosting as a custom upstream
